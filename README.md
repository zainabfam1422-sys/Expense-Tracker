# 💰 Expense Tracker

A simple expense tracking application to manage daily income and expenses.

## ✨ Features

- Add Expenses
- Delete Expenses
- Income & Expense Summary
- Total Balance
- Local Storage Support
- Responsive UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
